# CIVE202_Portfolio
This repo contains the files related to the 5 projects completed in Civil Engineering Analysis II in Spring of 2026.

Project 1 focused on impoorting and analyzing data related to air quality in Nebraska.  In doing so, the data had to be sorted and indexed, functions were made to make different data points categorical, and different types of pollution at different locations were examined.  This was useful in getting to know how to do more than just simple base-level tasks in Python and is related to the environmental portion of civil engineering.

Project 2 related to using Python to automate processes.  An outdated Excel file that was used to generate concrete mix designs was as the base for creating an open-source Python code that could do the same task, but more efficiently.  It was valuable in learning about the automation process while also very topical to civil engineering, as the use of concrete is important for structures, transportation, and more.

Project 3 was where simulations were introduced.  Traffic data was used to relate the position, speed, and acceleration  of a leader and follower vehicle.  This data was then used to simulate how a vehicle would react on the given route.  Creating simulations is an important and versatile skill in Python, and this specific one was related to transportation engineering.

Project 4 examined risks and hazards.  Data from the National Risk Index and from the Social Vulnerability Index was examined to create our own definition of risk and apply this to two hazards across two states.  This was to show how risk definitions and Python code may be biased, as there are key variations between our adjusted risk defintion and that of the NRI.  This project was also related to environmental engineering.

Project 5, which at this time is still ongiong, is also related to risks and hazards and is somewehat a continuation of Project 4.  It is more focused on generating map plots to visualize data.  Creating maps to display spacial data is important to show how data sets vary over given areas.
